# 🚛 Sistema de Transportadora — Interface Angular

Este é o repositório do Front-end do sistema de gerenciamento de transportadora. Desenvolvido em **Angular 18**, o projeto foi criado para consumir a API REST e fornecer uma interface moderna e intuitiva para o gerenciamento de dados.

## 📋 Sobre o Projeto
Interface web para gerenciar os módulos da transportadora, focada em produtividade e feedback visual para o usuário.

### ✅ O que foi feito (CRUD Administradores)
* **Listagem (Read):** Tabela dinâmica que consome dados da API em tempo real.
* **Atualização (Update):** Sistema de Pop-up para edição de dados sem recarregamento de página.
* **Feedback Visual:** Efeitos de interação nos botões e estilização responsiva.

### 🚀 O que o projeto vai ter
* **CRUD de Caminhoneiros:** Telas de cadastro, listagem e edição de motoristas.
* **Gestão de Fretes:** Controle e visualização das entregas.


## 🛠️ Tecnologias Utilizadas
* **Angular 18+**
* **TypeScript**
* **Tailwind CSS** (Estilização baseada em utilitários)
* **RxJS** (Consumo assíncrono de API)
* **HttpClient** (Comunicação com Back-end)
